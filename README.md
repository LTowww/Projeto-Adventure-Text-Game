# ⚔️ A Grande Aventura
Um jogo de aventura baseado em texto desenvolvido em **Portugol**, **Python** e **C**, onde o jogador precisa explorar florestas, cavernas, enfrentar criaturas e tomar decisões que mudam o rumo da jornada.

## 📖 Sobre o jogo
Em **A Grande Aventura**, você acorda em um lugar desconhecido e precisa escolher seus caminhos para sobreviver.
Durante a jornada, você poderá:
- 🌲 Explorar uma floresta misteriosa
- 🕳️ Entrar em cavernas escuras
- 👹 Enfrentar Goblins e Trolls
- 🧪 Encontrar poções de cura
- ⚔️ Participar de batalhas com dano aleatório
- 🏘️ Encontrar uma aldeia segura para finalizar sua aventura

Cada escolha altera o rumo da história.

---

## 🎮 Funcionalidades
- Sistema de escolhas
- Sistema de batalha por turnos
- Vida do jogador
- Inimigos com atributos diferentes
- Eventos aleatórios usando sorteio
- Caminhos alternativos
- Sistema de Game Over
- Estrutura modular utilizando funções

## ▶️ Como executar

### Versão Portugol
1. Abra o Portugol Studio
2. Crie um novo programa
3. Cole o código do jogo
4. Execute o projeto

### Versão Python
1. Certifique-se de ter o Python 3 instalado
   ```bash
   python --version
   ```
2. Clone o repositório
   ```bash
   git clone https://github.com/seu-usuario/a-grande-aventura.git
   cd a-grande-aventura
   ```
3. Execute o jogo
   ```bash
   python main.py
   ```

### Versão C
1. Certifique-se de ter o GCC instalado
   ```bash
   gcc --version
   ```
2. Clone o repositório
   ```bash
   git clone https://github.com/seu-usuario/a-grande-aventura.git
   cd a-grande-aventura/c
   ```
3. Compile o jogo
   ```bash
   gcc -o jogo jogo.c
   ```
4. Execute o jogo
   ```bash
   ./jogo.exe
   ```
